# **📊 Customer Churn Prediction – Regork Telecom**
Built an end-to-end data science project to analyze and predict customer churn using a dataset of 5,999 telecom customers. Performed EDA to identify key churn drivers across demographics, account details, and usage behavior, while filtering out noisy features.

Developed and evaluated an XGBoost model to predict churn, and generated predictions for a 1,000-customer test set. Delivered actionable insights to support targeted retention strategies and reduce customer attrition.
